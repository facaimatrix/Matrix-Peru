# MatrixPro Peru Release

This folder contains the final built installers for MatrixPro Peru version 1.0.0.

## Included files

- `MatrixPro Peru_1.0.0_x64_en-US.msi`
- `MatrixPro Peru_1.0.0_x64-setup.exe`

## How to share on GitHub

1. Create a GitHub repository or use an existing one.
2. Go to the repository's `Releases` tab.
3. Draft a new release (recommend version `v1.0.0`).
4. Upload the two installer files from this folder as release assets.
5. Add a short description and publish the release.

## Notes

- This package contains the final product only, not source code.
- If you want users to install directly, the `.msi` and `.exe` are the files to share.
